Early Access

สมาชิกทดสอบกลุ่มแรก

VelaFlow จะเริ่มทดสอบกับสมาชิกทดสอบกลุ่มแรก 5-10 คน ก่อนเปิดใช้งานในวงกว้าง

  • ทดสอบจาก workflow การทำงานจริงของครีเอเตอร์
  • พัฒนาต่อจาก feedback ของผู้ใช้กลุ่มแรก
  • สมาชิกทดสอบกลุ่มแรกได้รับสิทธิ์ใช้งานฟรี 1 ปี หากผลิตภัณฑ์พร้อมขายจริง
  • ออกแบบเพื่อครีเอเตอร์ โปรเจกต์เพลง และระบบคอนเทนต์

Roadmap

Roadmap ของ VelaFlow

แผนพัฒนาแบบค่อยเป็นค่อยไป จากระบบภายใน VelaLab สู่เครื่องมือที่ครีเอเตอร์ใช้งานได้จริง

Phase 1

Phase 1: VelaFlow Music

เริ่มจากระบบเพลง คอนเซปต์ เนื้อเพลง mood direction ไอเดียปก และพื้นฐานการปล่อยผลงาน

Phase 2

Phase 2: Motion / MV / Clips

ต่อยอดสู่ visual prompts, cinematic direction, storyboard และระบบคอนเทนต์สั้น

Phase 3

Phase 3: Live tools and creator automation

พัฒนา live tools, creator dashboards, workflow memory และ automation สำหรับงานสร้างสรรค์
